# ZEVA — Voice-First AI Assistant

> *"Hey ZEVA" — and she's there.*

ZEVA is a futuristic, voice-first AI assistant with a holographic floating orb UI. Built as a Progressive Web App (PWA) with wake word detection, real-time speech recognition, and Claude AI integration.

## Features

- 🎙️ **Wake word detection** — "Hey ZEVA" activates hands-free
- 🤖 **Claude AI** — intelligent, conversational responses
- 🌐 **PWA** — installable on Android & desktop
- 🔊 **Text-to-speech** — ZEVA talks back
- 💬 **Conversation memory** — context across turns
- 🎨 **Holographic orb UI** — sci-fi inspired interface

## Getting Started

```bash
git clone https://github.com/riiyansh-singh/ZEVA.git
cd ZEVA
```

Open `index.html` in Chrome. Web Speech API works best in Chrome/Edge.
